🚀 快速开始

# 1. 安装
git clone https://github.com/yourname/liquid-blog.git ~/ink
cd ~/ink
./install.sh

# 2. 初始化
ink init

# 3. 创建第一篇文章
ink new "Hello Liquid Blog"

# 4. 编辑文章
ink edit 2025/03/20-hello-liquid-blog/

# 5. 发布
ink publish 2025/03/20-hello-liquid-blog/ --channels blog,mastodon

# 6. 搜索
ink search "AI 开发"

# 7. 自然语言命令
ink "找出所有 TODO 并按优先级排序"
